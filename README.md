# To Do List Application (Python)

## Description

This project is a simple command line To Do List application written in Python.
It allows users to manage daily tasks by adding, viewing, completing, and removing tasks.
The program also saves tasks in a file so the list remains available when the program runs again.

## Features

* Add new tasks with a due date
* Categorize tasks (Personal, Work, Shopping, Wishlist)
* View all tasks in a clear format
* Mark tasks as completed
* Remove tasks from the list
* Filter tasks by category
* Display completed tasks separately
* Save tasks to a file and load them when the program starts

## How the Program Works

1. When the program starts, it loads tasks from the file `tasks.txt`.
2. The user is shown a menu with options to manage tasks.
3. Tasks can be added with a category and due date.
4. Users can mark tasks as completed instead of deleting them.
5. Tasks can also be filtered by category for better organization.
6. When the program exits, all tasks are saved back to the file.

## Menu Options

1. Add Task
2. View Tasks
3. Mark Task Completed
4. Remove Task
5. Show Completed Tasks
6. Filter by Category
7. Exit
